  • Create a new extraction job. This will generate a json file in config directory named config/config_job_name.json
  • Edit an existing extraction job. All the parameters can be modified except the job_name. Saved as a config json file
  • Copy an existing extraction job to create a new one. A new job_name is required. Saved as a new config json file
  • Run the extraction job. The start page let modify the execution parameters before running.
  • Delete extraction job. Delete the job from the database. The json file is not deleted
